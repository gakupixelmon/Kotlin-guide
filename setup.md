# Note: The setup guide is currently available in Japanese only.
まず環境構築をします。
VS Codeでも可能ですが最も推奨されるのはIntelliJ IDEAを使います。  
↓をクリックしたらwindowsかmacOSかLinuxかを選んで、その後青いボタンを押してダウンロードしましょう。  
[IntelliJ IDEAのダウンロードページ](https://www.jetbrains.com/ja-jp/idea/download/)  
古いサイトではこのボタンは有料版だと言われているかもしれませんが、現在は統合されているので無料版でも有料版でもここをクリックします。
