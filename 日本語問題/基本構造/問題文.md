fun 関数名(変数名1, 変数名2, ...) : 戻り値の型 {}

```

}
```

# 変数、型
```
fun main(){
    val a = 100
    var b = 100
    b = 200
}
```
Kotlinの変数には作り方が二つあります。
①一度値を入れたら変更できない変数。変数名の前にvalと書く。(valはvalueの略)
②値を入れた後も変更できる変数。変数名の前にvarと書く。（varはvariationの略）
valに後から違う値を入れようとすると、エラーが起きます。
バグを防ぐにはvalを使う方が良いため、なるべくvalをつかうよう心がけます。
変数には文字列を入れることもできます。先ほどの"Hello, World!"は文字列です。文字列は"で囲まれています。  
PythonやJavaScriptでは'で囲まれたものも文字列ですが、Kotlinでは"のみです。

valとvarはすでに学んだfunと同じような役割でキーワード的な役割をします。
別の役割を持つ単語として、IntやStringがあります。これらは型を宣言するのに使います。
整数はInt型、文字列はString型です。代表的な型を表にしておきます。これ以外にも非常にたくさんあるので各自調べましょう。
|32bit整数|64bit整数| 32bit少数 | 64bit少数 | 文字(1文字限定) | 文字列(1文字もそれ以上も) |
| ---- |----| ---- |----| ---- | ---- | 
| Int |Long| Float |Double| Char | String | 

関数の()の後の : 戻り値の型  はこのIntやStringのことを言っています。戻り値は関数の時に扱います。
変数を作る時、変数の型も決めることができます。(文字列は数字も可です。)
```
val a : String = "abc123"
```
しかし、Kotlinは型を自分で判断してくれるため書かなくても構いません。他の言語ではPythonも自分で判断してくれますが、
C言語は必ず型を決めなくてはいけません。(型を決めなくてはならないメリットもあって、実行が早く
簡単なので初歩的なコンピュータでも使えます。)

### 問題2
print関数は()に変数を入れると変数の値を表示します。変数fiveを作成して、print(five)とすることにより5を表示する
プログラムを作成しましょう
```
fun main(){
}
```

# 解答
問題1
```
fun main(){
    print("Hello, World!")
}
```

問題2
```
fun main(){
    val five = 5
    print(five)
}
```
